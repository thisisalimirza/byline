How to publish this article

Export the article in whichever format your platform needs.

Markdown includes the title as an H1 and the meta description as a blockquote. HTML wraps everything in a semantic <article> tag with the meta as a comment at the top.

  1. In WordPress, go to Posts → Add New
  2. Paste the title into the title field at the top of the editor
  3. In Gutenberg, add a Classic block and paste the article body — this preserves H2 headings and paragraph structure cleanly
  4. If you have Yoast SEO or Rank Math, scroll to the SEO panel and paste the meta description into the "Meta Description" field
  5. Set a featured image, assign a category, and add 3–5 tags related to the keyword
  6. Hit Preview to check formatting, then Publish

Tip: If headings aren't rendering correctly in Gutenberg, switch to the Code editor view and paste the HTML-formatted version instead.

  1. Open your Webflow CMS collection and create a new blog post item
  2. Paste the title into the Name / Title field
  3. Click into the Rich Text body field and paste the article — Webflow preserves heading levels and paragraph breaks automatically
  4. Go to the SEO Settings tab on the collection item and paste the meta description there
  5. Set a clean URL slug using the target keyword, lowercase and hyphenated (e.g. local-seo-small-business)
  6. Publish the item, then submit the URL to Google Search Console for faster indexing

If your Rich Text element uses custom styles in the Designer, make sure H2 and paragraph styles are defined in your global CMS styles — otherwise pasted headings may inherit unexpected styling.

  1. Ghost: Go to Posts → New Post and paste the title at the top
  2. Ghost supports Markdown natively — paste the Markdown version directly into the editor for the cleanest result
  3. Click the settings gear (⚙), expand Meta data, and paste the meta description plus a custom slug
  4. Substack: Create a new post, paste the title into the subject line, and paste the body into the editor
  5. For Substack, the title doubles as the email subject line — make sure it creates curiosity without being clickbait
  6. Add a header image and publish (or schedule for your audience's peak open time)

Ghost's Markdown editor renders the article live as you paste, so you can spot formatting issues before publishing.
